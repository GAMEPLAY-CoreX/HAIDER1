# لعبة مزاد الأحرف العربية

هذه لعبة بسيطة للمزايدة على الأحرف العربية. كل لاعب يدخل كلمة تحتوي على الحرف الذي يتم عرضه. في نهاية كل الجولات، اللاعب الذي جمع أكثر كلمات يفوز.

## المميزات

- يدعم من 2 إلى 10 لاعبين
- واجهة احترافية بلون الفضاء (فضاء النجوم)
- تأثيرات CSS بسيطة
- قاعدة بيانات SQLite لتخزين الجلسات والكلمات

## التشغيل

1. تأكد من وجود Node.js مثبتًا.
2. انتقل إلى المجلد `h` في الطرفية.
3. نفّذ:
   ```bash
   npm install
   npm start
   ```
4. افتح المتصفح إلى http://localhost:3000

## كيفية اللعب

- أدخل أسماء اللاعبين مفصولة بفواصل.
- اضغط "ابدأ اللعبة".
- سيتم عرض حرف عشوائي.
- أدخل كلمة تحتوي هذا الحرف واضغط "أرسل الكلمة".
- اضغط "الحرف التالي" للانتقال للحرف الجديد.
- بعد 10 حروف أو عند الانتهاء، ستُعرض النتائج.
